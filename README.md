Проект "Сложно сосредоточиться" (no focus)
📝 Описание проекта
Одностраничный сайт с адаптивной вёрсткой и переключением цветовых тем (тёмная/светлая/авто). Проект посвящён проблемам концентрации внимания и способам их преодоления.

🎯 Функциональность
Адаптивная вёрстка под три основных разрешения:

Мобильные устройства (375px)

Планшеты (768px)

Десктопы (1024px)

Переключение цветовых тем:

🌙 Тёмная тема (Neon)

☀️ Светлая тема (Day)

🤖 Автоматическая тема (следует системным настройкам)

Фиксированный фон с эффектом parallax

Декоративные элементы (уголки, REC-индикатор)

Адаптивная галерея с сеткой изображений

Семантическая вёрстка с использованием HTML5 тегов

🛠 Технологии
HTML5

CSS3

Grid Layout

Flexbox

CSS-переменные (custom properties)

Медиазапросы

Логические свойства (margin-block-start, inset-inline-end и др.)

JavaScript (переключение тем)

🎨 Особенности вёрстки
Цветовые схемы
Тёмная тема:

Акцентный цвет: #FF0070

Фон: #000028

Текст: #F1B2CE

Заголовки: #DB00FF

Светлая тема:

Акцентный цвет: #FF8DCB

Фон: #FFF1F7

Текст: #353430

Заголовки: #FFC2E6

Адаптивные размеры текста
Использован clamp() для плавного изменения размеров:

css
/* Для мобильных */
clamp(7.25rem, 7.0115rem + 1.0178vw, 7.5rem)

/* Для планшета и десктопа */
clamp(7.5rem, 0.5625rem + 14.4531vw, 9.8125rem)
Ключевые особенности
Фиксированный фон: background-attachment: fixed

Ленивая загрузка изображений в галерее

Декоративные уголки через псевдоэлементы

Доступность: скрытие декоративных элементов от скринридеров

Переменные для тем: единые имена переменных для обеих тем

📱 Адаптивные брейкпоинты
Устройство	Ширина	Особенности сетки
Мобильное	< 768px	Вертикальное расположение, одна колонка
Планшет	768px - 1023px	Смешанная сетка, 2-3 колонки
Десктоп	≥ 1024px	Горизонтальное меню, 3 колонки
